DESCRIPCIÓN
Este proyecto tiene como objetivo desarrollar y demostrar habilidades prácticas en SQL y análisis de datos, utilizando un conjunto de datos reales sobre el clima de la ciudad de Chicago en 2017. El enfoque principal es la exploración, limpieza y análisis de datos mediante 
consultas SQL, complementadas con Python y Jupyter Notebook.

El proyecto simula un flujo de trabajo típico de análisis de datos, desde la obtención del dataset hasta la generación de conclusiones a partir de los datos.

TECNOLOGÍAS UTILIZADAS
- SQL para consultas de exploración y análisis de datos
- Python para manipulación y análisis.
- Jupyter Notebook para la documentación y visualización del proyecto.
- Git/Github para el control de versiones.
